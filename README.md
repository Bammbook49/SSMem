# SSMem

Простая аркадная игра в стиле «космического шутера» на Python с использованием библиотеки Pygame.

## Описание

В игре вы управляете космическим кораблём, который должен сбивать вражеские корабли и уворачиваться от астероидов. За каждого сбитого врага начисляются очки, но если пропустить слишком много врагов — игра заканчивается.

## Игровой процесс

* Управляйте кораблём с помощью клавиш **A** (влево) и **D** (вправо).
* Стреляйте по врагам с помощью **Пробела**.
* Сбивайте вражеские корабли, чтобы набирать очки.
* Избегайте столкновений с врагами и астероидами.
* Не пропускайте слишком много врагов: если количество пропущенных превысит счёт на 10 — вы проиграете.

## Правила победы и поражения

* **Победа**: невозможно проиграть, игра продолжается до столкновения или пропуска слишком большого количества врагов.
* **Поражение**:
    * столкновение с вражеским кораблём или астероидом;
    * количество пропущенных врагов превышает счёт на 10 и более.

Особенности игры
Динамическое появление врагов: за каждого сбитого врага появляется новый.

Система столкновений: игра проверяет столкновения игрока с врагами и астероидами.

Визуальные уведомления о поражении и перезапуске.

Звуковое сопровождение при поражении.

Отображение счёта и количества пропущенных врагов в реальном времени.

Классы и их назначение
GameSprite — базовый класс для всех игровых объектов (наследует pygame.sprite.Sprite).

Player — класс игрока, реализует управление и стрельбу.

Enemy — класс вражеских кораблей, движется сверху вниз.

Bullet — класс пуль, выпущенных игроком, движется вверх.

Asteroid — класс астероидов, движется сверху вниз, наносит урон при столкновении.
